# PROJETO 01 – IMPLEMENTAÇÃO DE SISTEMA DE LOGS:
#### Implemente um sistema de logging utilizando o padrão Chain of Responsibility, onde diferentes manipuladores de log tratam mensagens com base em seu nível de severidade:
- LoggerConsole: Trata logs do nível INFO.
- LoggerArquivo: Trata logs dos níveis INFO e WARNING.
- LoggerEmail: Trata logs do nível ERROR.
#### Tarefas:
- Crie as classes correspondentes em uma linguagem de sua escolha (JavaScript, Python ou Java).
- Configure a cadeia de responsabilidade de forma que cada log seja tratado pelo(s) manipulador(es) apropriado(s).
- Teste o sistema enviando logs de diferentes níveis e observe quais manipuladores os processam
